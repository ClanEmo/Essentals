# Essentials

포켓마인버전 에센셜입니다.

## Commands

※ Default
* `/home [name]`
* `/sethome <name>`
* `/delhome <name>`
* `/mute <playername>`
* `/back`
* `/tree <tree|birch|redwood>`
* `/setspawn`
* `/burn <playername> <second>`
* `/kickall [reason]`
* `/killall [reason]`
* `/heal [playername]`
* `/clearinventory [playername] [item id]`

※ Login
* `/login <password>`
* `/register <password> <password>`
* `/unregister <password>``
* `/changepassword <oldpassword> <newpassword>`
* `/logout`
